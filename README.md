Lab 3: Exploratory Data Analysis (EDA) – Netflix Dataset
Dataset

Netflix Movies and TV Shows Dataset
Source: Kaggle – https://www.kaggle.com/datasets/shivamb/netflix-shows

Dataset Size: 8,807 titles
Features: 12 columns

Problem Type

Exploratory Data Analysis (EDA)

Problem Statement

The goal of this lab is to perform Exploratory Data Analysis (EDA) on the Netflix dataset to understand the structure, patterns, and relationships within the data.

The analysis focuses on identifying content distribution, missing values, country contributions, rating patterns, and trends in content release and addition over time.

Description

This dataset contains information about Netflix movies and TV shows, including title, type, director, cast, country, release year, rating, duration, and genre.

EDA was performed to:

Clean and preprocess the data

Handle missing values

Analyze content distribution

Identify top producing countries

Examine rating distribution

Explore time-based trends

The results provide insights into Netflix’s content strategy and growth over the years.
